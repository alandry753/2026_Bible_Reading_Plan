# 2026 Bible Reading Plan

A 54-week, historically grounded Bible reading plan integrating:

- Torah (Ashkenazic weekly aliyot, daily splits)
- Tanakh narrative blocks
- Chronological New Testament
- Psalms (thematic / stanza-based)
- Proverbs (bite-sized daily wisdom)

Includes:
- Mobile-friendly web interface
- Daily + monthly views
- Search and version selection
- Optional JPS Tanakh linking
